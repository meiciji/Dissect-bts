Dissect Reverse Engineering the Web

A guided web development project for curious coders who want to take apart and rebuild the web.

üí° Concept / Overview

Dissect is where high schoolers pick an existing website (Spotify, Notion, Pinterest, Wordle, etc.) and "reverse engineer" a simplified version using HTML, CSS, and JavaScript. Through this creative process, participants learn how modern web tools work ‚Äî and remix them into something all their own.
